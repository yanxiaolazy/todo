## TODO

### Installation

```bash
$ git clone https://github.com/yanxiaolazy/todo
```

### Development

安装依赖项

```bash
$ yarn install
```

启动客户端

```bash
$ yarn start
```

初始化数据库

```bash
$ yarn run db
```

注：确保连接到一个有效的数据库

启动服务端

```bash
$ yarn run dev
```

### Production

安装依赖项

```bash
$ yarn install
```

打包项目

```bash
$ yarn run build
```

开启服务

```bash
$ docker-compose up -d --build
```

**每次更新项目后都需要重复这个过程**
